Beautiful R Plots

2020-08-16

The R language is fantastic for quickly turning data into plots, however, it is not commonly used for creating visual aids for presentation or publication figures. While it is more common to produce final figures with other software, the same quality figures can be created in R.

Herein I show several examples of plots, that I have re-created using R, in an aesthetically similar manner. In each example, I provide the original figure and then the R-code I used for replication, followed by the replicate figure. While I am exploring packages like waffle and ggplot2, there are other packages could certainly achieve similar outcomes.

I would love to hear any recommendations on possible improvements in the comments or @gryk_h on twitter.

Example 1

This example is from the wikipedia page Pie Chart.

This is the method used to reproduce the figure:

pubs <- c(
  `E` = 16, `D` = 36, `C` = 15, `B` = 32, `A` = 1)
waffle(
  pubs , rows = 10, size = 0.5, legend_pos = "bottom", flip = TRUE,
  colors = c("darkred", "darkblue", "darkgreen", "orange", "pink"))

Example 2

This image was found with a quick google search query gun homicides plot. There are lots of variations out there.

This is the method used to reproduce the figure (with minor aesetic upgrades):

dat <- tibble(country = toupper(c("US", "France", "Canada", "Germany", "Australia", "UK", "Japan")),
              count = c(11.2, 2.4, 1.9, 1.0, 0.9, 0.2, 0),
              label = c(as.character(c(11.2, 2.4, 1.9, 1.0, 0.9, 0.2, 0))),
              code = c("us", "fr", "ca", "de", "au", "gb", "jp"))

dat %>% mutate(country = reorder(country, -count)) %>%
  ggplot(aes(country, count, label = label)) +
  geom_bar(stat = "identity", fill = c("darkred", "darkblue", "darkblue", "darkblue", "darkblue","darkblue", "darkblue")) +
  geom_text(nudge_y = 0.5, color = "black", size = 5) +
  geom_flag(y = -1, aes(country = code), size = 12) +
  scale_y_continuous(breaks = c(0, 2, 4, 6, 8, 10, 12), limits = c(0,12)) +   
  ggtitle(toupper("Gun Death Rate Per 100,000")) + 
  xlab("") + 
  ylab(" ") +
  ggthemes::theme_economist_white(gray_bg = FALSE) +
  theme(axis.text.x = element_text(size = 8, vjust = -16),
        axis.ticks.x = element_blank(),
        axis.line.x = element_blank(),
        plot.margin = unit(c(1,1,1,2), "cm"))

Example 3

This plot was found on ResearchGate in the following Review. Alternate data was used to produce this plot.

This is the method used to reproduce the figure, although a random dataset was used to generate the data, so slight differences in the points should be expected:

hypFun <- dat.hackshaw1998 
res <- rma(yi, vi, data=hypFun, measure="OR", method="FE")
funnel(res, xlab = 'Observed Outcome')
